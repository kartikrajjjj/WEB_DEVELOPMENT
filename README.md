# Web Development Projects 🚀

This repository contains hands-on web development projects built using **HTML, CSS, and JavaScript**.
The focus is on practical implementation, DOM manipulation, and building functional user interfaces.

---

## 🔧 Tech Stack
- HTML5  
- CSS3  
- JavaScript (ES6)  
- Browser APIs (DOM, localStorage)

---

## 📁 Projects Included

### 🛒 Myntra Website Clone (Fully Functional)
A fully functional front-end clone of the Myntra e-commerce platform built using HTML, CSS, and JavaScript.  
Features include product listing, dynamic cart management, filtering, and state handling using `localStorage`.

---

### ✅ Todo App
A task management application that allows users to add, delete, and manage daily tasks.  
Uses JavaScript DOM manipulation and `localStorage` to persist tasks across browser sessions.

---

### 🧮 Basic Calculator
A simple and responsive calculator supporting basic arithmetic operations.  
Built using JavaScript for logic handling and DOM updates for real-time display.

---

### ✊✋✌ Stone Paper Scissors Game
An interactive browser-based game where users play against the computer.  
Implements game logic, random choice generation, and dynamic result display using JavaScript.

---

- UI development using HTML & CSS
- Managing application state using localStorage
- Writing clean and structured frontend code
